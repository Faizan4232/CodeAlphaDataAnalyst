# 2. Task 2 - Exploratory Data Analysis (EDA)

In this task, I explored the Titanic dataset and found useful insights from the data.

Analysis Performed:
- Viewed first rows of dataset
- Checked dataset shape
- Checked column data types
- Found missing values
- Viewed summary statistics
- Calculated survival rate
- Compared survival by gender
- Analyzed age and fare columns
- Created correlation heatmap

Main Insights:
- Female passengers had higher survival rate than male passengers
- First class passengers had better survival chances
- Some columns had missing values
- Fare and passenger class affected survival
